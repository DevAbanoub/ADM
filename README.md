# sidm
Smart Internet Download Manager

NOTE : This App Is UNDER DEVELOPMENT now!

## Goals || Features
- use concurrency to download files
- easy to use (just a click to download any file)
- use CLI or GUI
- cross-platform (GNU/Linux, Mac, Windows, Android)
